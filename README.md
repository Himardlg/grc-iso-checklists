## 🎯 ¿Para qué sirve?

Estas checklists permiten a cualquier organización:

- **Evaluar su nivel de madurez** en cada control de las normas ISO
- **Identificar GAPs** respecto a los requisitos del estándar
- **Priorizar acciones de mejora** ordenando por puntuación
- **Documentar evidencias** disponibles y asignar responsables
- **Preparar auditorías** internas o de certificación
- **Construir un roadmap GRC** basado en datos objetivos

---

## 📊 Escala de madurez (0–4)

| Nivel | Nombre | Descripción |
|---|---|---|
| **0** | No definido | No existe ningún proceso, control o documentación |
| **1** | Inicial | Existe algo pero de forma informal o sin documentar |
| **2** | Parcial | Documentado pero incompleto o aplicado de forma inconsistente |
| **3** | Definido | Completamente documentado, implementado y operativo |
| **4** | Optimizado | Medido, revisado y mejorado de forma continua |

---

## 🚀 Cómo usar las checklists

1. **Descarga** el archivo Excel de la norma que quieras evaluar
2. **Abre** la pestaña `📋 Instrucciones` para leer la guía
3. **Ve** a la pestaña `Checklist` y expande cada dominio
4. **Puntúa** cada control del 0 al 4 usando el desplegable
5. **Documenta** la evidencia actual que tienes disponible
6. **Asigna** responsable y fecha objetivo para los controles con puntuación baja
7. El **% de madurez** por dominio se calcula automáticamente

> 💡 **Tip:** Filtra la columna "Puntuación" por 0 o 1 para identificar los controles más críticos y priorizarlos en tu plan de acción.

## 📁 Estructura de cada checklist

Cada archivo Excel incluye dos pestañas:

**`📋 Instrucciones`**
- Guía de uso paso a paso
- Escala de madurez explicada
- Contexto y descripción de la norma

**`Checklist [ISO]`** — Todos los controles organizados por dominio con:

| Columna | Descripción |
|---|---|
| ID | Identificador del control según la norma |
| Control / Requisito | Nombre y descripción del control |
| Evidencia requerida | Qué documentos o registros deberías tener |
| Puntuación (0-4) | Desplegable para evaluar el nivel de madurez |
| Evidencia actual | Lo que ya tienes disponible hoy |
| Responsable | Dueño del control en tu organización |
| Fecha objetivo | Plazo para alcanzar el nivel deseado |
| Observaciones / GAPs | Notas libres, decisiones o brechas identificadas |

## 🏛️ Sobre las normas

### 🤖 ISO/IEC 42001:2023 — Gestión de IA
El **primer estándar internacional** para implementar un Sistema de Gestión de IA (SGAI). Publicado en diciembre de 2023, cubre gobernanza, gestión de riesgos, transparencia, ética y responsabilidad en el uso de IA. Aplicable a cualquier organización que desarrolle, provea o use sistemas de IA.

### 🔒 ISO/IEC 27001:2022 — Seguridad de la Información
Estándar internacional para la gestión de la seguridad de la información (SGSI). La versión 2022 actualiza el Anexo A con **93 controles** en 4 dominios: organizativos (A.5), personas (A.6), físicos (A.7) y tecnológicos (A.8).

### 🔄 ISO 22301:2019 — Continuidad de Negocio
Estándar para implementar un Sistema de Gestión de Continuidad de Negocio (SGCN). Incluye requisitos para el análisis de impacto (BIA), estrategias de continuidad, planes BCP/DRP/IRP y programas de ejercicios y pruebas.
